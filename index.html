// 1. HÀM CHUYỂN ĐỔI AN TOÀN TUYỆT ĐỐI: Biến mọi thứ thành chuỗi viết thường, không dấu, xóa rác function cleanKeySafe(cellValue) { if (cellValue === undefined || cellValue === null) return ''; // Nếu là ô trống, trả về chuỗi rỗng chứ không để undefined // Ép cứng thành chuỗi, loại bỏ dấu tiếng Việt, loại bỏ khoảng trắng và ký tự xuống dòng (Alt+Enter) return String(cellValue) .toLowerCase() .normalize('NFD') .replace(/[\u0300-\u036f]/g, '') .replace(/[^a-z0-9]/g, ''); // Chỉ giữ lại chữ và số cơ bản } // 2. ENGINE ĐỌC EXCEL KHÔNG THỂ BỊ SẬP async function xuLyExcelNhanSuThongMinh(input) { if (!input.files || input.files.length === 0) return; const file = input.files[0]; const reader = new FileReader(); reader.onload = async function(e) { try { const data = new Uint8Array(e.target.result); const workbook = XLSX.read(data, { type: 'array', cellDates: true }); const sheet = workbook.Sheets[workbook.SheetNames[0]]; const rows = XLSX.utils.sheet_to_json(sheet, { header: 1 }); if (rows.length === 0) { alert("File Excel này trống rỗng!"); return; } // BƯỚC 1: DÒ HÀNG TIÊU ĐỀ AN TOÀN (Vòng lặp thô, không dùng includes trực tiếp) let headerRowIndex = -1; let headers = []; for (let i = 0; i < rows.length; i++) { let rowArr = rows[i] || []; // Làm sạch toàn bộ hàng này, biến undefined thành chuỗi "" hết! let cleanedRow = rowArr.map(cell => cleanKeySafe(cell)); // Bây giờ tất cả phần tử trong cleanedRow chắc chắn là String, kiểm tra an toàn 100% let coCotHoTen = false; for (let j = 0; j < cleanedRow.length; j++) { if (cleanedRow[j] === 'hovaten' || cleanedRow[j] === 'hoten' || cleanedRow[j].indexOf('hoten') !== -1) { coCotHoTen = true; break; } } if (coCotHoTen) { headerRowIndex = i; headers = cleanedRow; // Lưu lại mảng tiêu đề đã được làm sạch tinh khiết break; } } if (headerRowIndex === -1) { alert("Hệ thống không tìm thấy cột 'Họ và tên' hoặc 'Họ tên' trong file. Vui lòng kiểm tra lại dòng tiêu đề!"); return; } // BƯỚC 2: ĐỊNH VỊ VỊ TRÍ CỘT AN TOÀN (Dùng indexOf trên mảng đã làm sạch) let colIdx = { hoTen: headers.findIndex(h => h === 'hovaten' || h === 'hoten' || h.indexOf('hoten') !== -1), ten: headers.indexOf('ten'), bhxh: headers.findIndex(h => h.indexOf('bhxh') !== -1 || h.indexOf('baohiem') !== -1), cccd: headers.findIndex(h => h.indexOf('cccd') !== -1 || h.indexOf('cmnd') !== -1), atv: headers.findIndex(h => h.indexOf('antoan') !== -1), nndh: headers.indexOf('nndh'), namVeHuu: headers.findIndex(h => h.indexOf('vehuu') !== -1), maDv: headers.findIndex(h => h.indexOf('madonvi') !== -1 || h.indexOf('donvi') !== -1), nam: headers.indexOf('nam'), namSinh: headers.findIndex(h => h.indexOf('namsinh') !== -1), nu: headers.findIndex(h => h.indexOf('nu') !== -1) }; // BƯỚC 3: TRÍCH XUẤT DỮ LIỆU VÀ ÉP KIỂU SỐ NĂM (Fix lỗi bẫy ngày tháng của file Book1) let dataRows = rows.slice(headerRowIndex + 1); let mapped = dataRows.map(row => { if (!row || row.length === 0 || colIdx.hoTen === -1 || !row[colIdx.hoTen]) return null; let hoTen = String(row[colIdx.hoTen]).trim(); if (hoTen === '' || hoTen.toLowerCase() === 'stt' || cleanKeySafe(hoTen) === 'hovaten') return null; let tenRieng = colIdx.ten !== -1 && row[colIdx.ten] ? String(row[colIdx.ten]).trim() : ''; if (!tenRieng) tenRieng = hoTen.split(' ').pop(); let bhxh = colIdx.bhxh !== -1 && row[colIdx.bhxh] ? String(row[colIdx.bhxh]).trim() : ''; let cccd = colIdx.cccd !== -1 && row[colIdx.cccd] ? String(row[colIdx.cccd]).trim() : ''; // Xử lý cột Năm (Ép cứng về kiểu Số nguyên) let namHienTai = 2026; if (colIdx.nam !== -1 && row[colIdx.nam] !== undefined && row[colIdx.nam] !== null) { let cellVal = row[colIdx.nam]; if (cellVal instanceof Date) { namHienTai = cellVal.getFullYear(); // Nếu Excel tự nhận là ngày tháng, lấy năm } else { let strVal = String(cellVal).trim(); let parts = strVal.split('-'); let yr = parseInt(parts[0]); if (!isNaN(yr)) { namHienTai = yr; } else { let parsed = parseInt(strVal.replace(/[^0-9]/g, '')); if (!isNaN(parsed)) namHienTai = parsed; } } } // Định dạng ngày tháng cho các trường ngày let formatNgay = (val) => { if (!val) return null; if (val instanceof Date) return `${val.getFullYear()}-${String(val.getMonth() + 1).padStart(2, '0')}-${String(val.getDate()).padStart(2, '0')}`; let s = String(val).trim(); return s.match(/^\d{4}-\d{2}-\d{2}/) ? s.substring(0, 10) : s; }; let namVeHuu = colIdx.namVeHuu !== -1 ? formatNgay(row[colIdx.namVeHuu]) : null; let namSinh = colIdx.namSinh !== -1 ? formatNgay(row[colIdx.namSinh]) : null; let nuVal = colIdx.nu !== -1 && row[colIdx.nu] ? cleanKeySafe(row[colIdx.nu]) : ''; let gioiTinh = (nuVal === 'x' || nuVal === '1' || nuVal === 'true') ? "Nữ" : "Nam"; let laAtv = colIdx.atv !== -1 && row[colIdx.atv] ? ['x','1','true'].includes(cleanKeySafe(row[colIdx.atv])) : false; let laNndh = colIdx.nndh !== -1 && row[colIdx.nndh] ? ['x','1','true'].includes(cleanKeySafe(row[colIdx.nndh])) : false; let maDv = colIdx.maDv !== -1 && row[colIdx.maDv] ? String(row[colIdx.maDv]).trim() : ''; let chucDanh = String(row[row.length - 1] || '').trim(); return { ho_va_ten: hoTen, ten: tenRieng, so_so_bhxh: bhxh, so_cccd: cccd, an_toan_vien: laAtv, nndh: laNndh, nam_ve_hu: namVeHuu, ma_don_vi: maDv, nam: parseInt(namHienTai) || 2026, // Chắc chắn là số nguyên nam_sinh: namSinh, gioi_tinh: gioiTinh, chuc_danh_cong_viec: chucDanh }; }).filter(x => x !== null); if (mapped.length === 0) { alert("Không tìm thấy dữ liệu nhân sự hợp lệ!"); return; } // Gửi dữ liệu lên Cloud const { error } = await client.from('nhan_su').insert(mapped); if (error) { alert("Database từ chối lưu dữ liệu: " + error.message); } else { alert(`🎉 Thành công dứt điểm! Đã đồng bộ ${mapped.length} nhân sự lên Cloud.`); await taiDuLieuHeThong(); } } catch (err) { alert("Lỗi hệ thống: " + err.message); } }; reader.readAsArrayBuffer(file); }
